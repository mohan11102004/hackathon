UI/UX Design Course

Course Description

This course covers the fundamentals of User Interface (UI) and User Experience (UX) design. Students will learn principles, methodologies, and tools used in designing effective and user-friendly digital interfaces.

Course Topics

  • Introduction to UI/UX Design
  • Understanding User Needs
  • Wireframing and Prototyping
  • Color Theory and Typography
  • Visual Hierarchy and Layout
  • Interaction Design
  • User Research and Testing
  • Responsive Design
  • Usability Principles

Course Schedule

  • Week 1: Introduction to UI/UX Design
  • Week 2: Understanding User Needs
  • Week 3: Wireframing and Prototyping
  • Week 4: Color Theory and Typography
  • Week 5: Visual Hierarchy and Layout
